## 📖 Project Overview

This repository contains the source code for the official brochure website of **BB&S Building Services Pty Ltd**.

The purpose of this website is to establish a professional online presence for the company by showcasing its services, completed projects, company information, certifications, and contact details. The website is designed to be modern, responsive, and easy to navigate across desktop, tablet, and mobile devices.

---

## 🎯 Project Objectives

- Create a professional online presence
- Showcase construction services
- Display completed projects
- Increase customer enquiries
- Build trust through certifications and company information
- Ensure a responsive user experience

---

## ✨ Features

- Responsive design
- Modern landing page
- About Us section
- Services overview
- Project gallery
- Contact form
- Mobile-friendly navigation
